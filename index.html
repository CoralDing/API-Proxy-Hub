使用方式

在域名后添加 /https-proxy/ 或 /http-proxy/，后接目标地址（不含协议部分）。我们的智能路由系统将自动处理请求转发。

URL 格式
HTTPS 请求: https://your-domain/https-proxy/api.example.com/v1/path
HTTP 请求: https://your-domain/http-proxy/example.com/path

代码示例

JavaScript
fetch('https://your-domain/https-proxy/api.example.com/v1/endpoint', {
    method: 'POST',
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        // 你的请求参数
    })
})
.then(res => res.json())
.then(data => console.log(data));

将 https://your-domain 替换为实际域名，将 api.example.com/v1/endpoint 替换为目标 API 地址。所有请求头和参数将被自动转发。
