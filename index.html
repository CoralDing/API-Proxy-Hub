API Proxy Hub

通用、高效的 API 代理服务

欢迎使用我们的 API 代理服务，支持 HTTP 和 HTTPS 请求，简化您的开发流程。

使用方式

在域名后添加 /https-proxy/ 或 /http-proxy/，后接目标地址（不含协议部分）。

HTTPS 示例: https://your-domain/https-proxy/api.example.com/v1/path
HTTP 示例: https://your-domain/http-proxy/example.com/path

代码示例

fetch('https://your-domain/https-proxy/api.example.com/v1/endpoint', {
    method: 'POST',
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        // 你的请求参数
    })
}).then(res => res.json()).then(data => console.log(data));

将 https://your-domain 替换为实际域名，将 api.example.com/v1/endpoint 替换为目标 API 地址。
